# 後記

或許看完歌詞後，有些人會覺得這是在講關於愛情的故事，但其實帶入從小一起玩的朋友或許更有感覺。
就像是每個人都有的經歷一樣，或多或少都有幾個曾經很要好的朋友，「從什麼時候開始的」，就這樣的不在聯絡了？
